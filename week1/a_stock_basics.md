# A股入门笔记：

## 1. A股

- 在中国境内上市、主要用人民币买卖的股票。
- 目前主要在上海证券交易所、深圳证券交易所和北京证券交易所交易。

- 买入一家公司的股票，就相当于买下了这家公司很小的一部分，成了它的股东。

- 股票投资能不能赚钱，主要看下面几件事：

  - **股价变化**：卖出价比买入价高，就赚了价差；反过来就是亏损。

  - **现金分红**：公司赚了钱，拿出一部分现金分给股东。

  - **送股或转增**：手里的股票数量变多了，但这不是白捡一笔钱，因为股价通常会跟着除权调整。


- 长期来看，股价要靠公司的利润和现金流撑着
- 短期复杂一些，市场情绪、政策消息、资金进出都会影响价格。 

## 2. A股市场的主要板块

| 板块     | 交易所         | 常见代码特征                   |
| -------- | -------------- | ------------------------------ |
| 沪市主板 | 上海证券交易所 | `600`、`601`、`603`、`605`开头 |
| 深市主板 | 深圳证券交易所 | `000`、`001`、`002`开头        |

创业板/科创板/北交所   都需要两年的交易经验+资产  （普通人就是沪市主板和深市主板）

## 3. 基本交易规则

- A股交易时间都是周一到周五（法定节假日排除）
- 时段：
  - 09:15-09:25  (盘前集合竞价)
    - 前五分钟可挂单可撤单
    - 后五分钟只能挂单
    - 09:25出开盘价
  - 09:30-11:30       13:00-14:57  （盘中连续交易）
  - 14:57-15:00  （尾盘集合竞价）
    - 可挂单不可撤单
  - 创业板和科创板15:05-15:30 可以收盘价成交
  - 夜市委托时间 不同券商不一样 一般18点到点
- 数目规则
  - 一手=100股
  - 主板股票最少买一手，必须买整数手（新手买不起太贵的比如贵州茅台）
- 回转交易规则
  - T+1：今天买的明天（下一个交易日）才能卖。当天买出股票的钱到资金账户也得明天才能转到银行卡
- 成交原则
  - 价格优先 时间优先
  - 在连续交易阶段  要是你的委托价格比别人好就能先成交，同样价格按照时间先后成交。
  - 涨停价买入和跌停价卖出  都会立刻成交（冲天炮和核按钮）
- 涨跌幅
  - A股跟国外的股票不同
  - 主板股票：上一个交易日收盘价的±10%
  - 科创板创业板 20
  - 北交所 30

## 4. 指数

指数像一个“班级平均分”：它挑出一组股票，用一个数字告诉你这批股票整体表现怎么样。

| 指数 | 主要含义 | 观察用途 |
| --- | --- | --- |
| 上证指数 | 上海证券交易所上市股票的综合表现 | 观察沪市整体情况 |
| 深证成指 | 深圳市场代表性公司的表现 | 观察深市核心公司 |
| 创业板指 | 创业板代表性公司的表现 | 观察成长风格 |
| 科创50 | 科创板代表性公司的表现 | 观察硬科技板块 |
| 沪深300 | 沪深市场中规模大、流动性较好的300只证券 | 常作大盘股基准 |
| 中证500 | 剔除沪深300成分后较有代表性的中盘公司 | 观察中盘风格 |
| 中证1000 | 更偏小市值的一组代表性公司 | 观察小盘风格 |

- 指数上涨不代表所有股票都涨。

- 有时只是几只权重股把指数拉上去了。









- 白马股：回报率高 具有投资价值 业绩优良 （医药 消费 科技 金融行业）（恒瑞医药 贵州茅台 腾讯控股 中国平安）
- 蓝筹股：长期稳定增长 大型的 传统工业股以及金融股 
- 成长股：增长快 成长性强
- 板块：根据某些特性分类
  - 行业板块
  - 概念板块（能扯上关系）
- 龙头股：对同行业同板块的其他股票具有号召力
- 赛道股：政策支持 高成长
- 题材股：特殊时段/重大社会事件 （口罩/疫苗/抗原）

## 6. K线

- 记录当天四个价格：

  - **开盘价**：当天第一笔或开盘集合竞价形成的价格。

  - **最高价**：当天成交的最高价格。

  - **最低价**：当天成交的最低价格。

  - **收盘价**：当天最后阶段形成的收盘价格。




- 粗的矩形是实体
- 上面的细线是上影线 最高价
- 下面  下影线   最低价



## 7. 常用技术指标

- 均线 MA（N）   （简单移动平均线）
  - MA(N) = 最近N日收盘价之和 / N
  - 股价在长期均线下方：趋势弱
  - 均线的金叉：MA5上穿MA20   
- MACD（指数平滑异同移动平均线）  默认参数（12，26，9）
    - 快线DIF 慢线DEA 能量柱 0轴
    - DIF = 12 EMA - 26 EMA       (EMA 均线是指数移动平均线，近期的价格分给更高的权重)
    - DEA = 快线的移动平均线 周期9
    - 能量柱=n*(DIF-DEA)   （快线和慢线的差离值）
      - 12EMA上穿26EMA，DIF=0， (DIF)'>0  
      - 12EMA下穿26EMA，DIF=0， (DIF)'<0  
    - DIF上穿DEA 形成金叉 多头信号   
      - （多头：认为会涨，买入的人/资金）（买方愿意不断提高报价抢货，买盘主动）
    - DIF下穿DEA 形成死叉 空头信号
- RSI   相对强弱指标 （0～100） （市场情绪温度计）
  - 上涨的力量强 向100靠近 。  反之向0靠近
  - 市场习惯：
    -    RSI>70  超买
    -    RSI<30  超卖
- 成交量
  - 放量：成交量成倍放大
  - 缩量：成交量成倍缩小



## 8. 基本面指标

- PE 市盈率 
  - 公司市值/一年净利润
  - 越高说明市场对未来预期高
  - 越低说明市场觉得一般/有风险
- PB（市净率）
  - 公司市值/净资产
  - 有多少家底
- ROE（净资产收益率）
  - 公司利用股东投入资金赚钱的能力
  - 净利润/股东投入资金
- 毛利率
  - 高通常说明产品有品牌优势、技术优势或者竞争优势
  - 高端白酒、奢侈品、软件公司的毛利率往往很高
  - 钢铁 水泥、煤炭这些竞争激烈的大宗商品行业毛利率通常较低。



### 9. 资金面指标 （研究钱往哪里流）

- 北向资金
  - 香港和海外资金通过沪股通、深股通买A股
  - 这些资金很多来自国际机构、外资基金、主权基金，所以市场经常把它当作“外资风向标”
- 融资融券
  - 融资：问券商借钱买股票
  - 融券：借股票做空（问券商借100股卖出，等跌了买回来还给券商）
  - ？有点像合约杠杆？
  - A股中融资比融券活跃
  - 看融资就能看市场情绪 看涨or not
- 主力资金
  - 资金规模明显大于普通散户的大资金，例如公募基金、私募基金、保险资金、社保基金、券商自营盘、游资、大机构等
