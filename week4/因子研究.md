## 基础概念

- 什么是因子？
  - 一种给股票打分的方式
  - 看股票的时候，股票最近很强/公司盈利不错/股票波动太大  这些概念太**主观**
  - 量化因子就是把这些主观的判断变成数字——可以量化的选股理由
- 因子有两个方向
  - 有的因子越高越好：ROE 毛利率 营收增长率 20日动量 北向资金流入
  - 有的越低越好：PE PB  波动率 最大回撤
- 多因子？
  - 比如网购不仅仅只看一个指标（销量/价格/差评率/发货速度）
  - 股票也如此，只看一个指标容易走偏
- 常见因子类型
  - 动量：最近走势强不强
    - 通常正向
  - 价值：现在贵不贵
    - PE 市盈率/PB 市净率/PS 市销率/股息率
    - 通常反向，但是便宜也可能是公司基本面差
  - 质量：赚钱能力和经营质量咋样
    - ROE ROA 毛利率 净利率 经营现金流 资产负债率等
  - 成长：增长快不快
    - 营收增长率/净利润增长率/扣非净利润增长率/经营现金流增长率
  - 波动：稳不稳，风险咋样
    - 收益率标准差/最大回撤
  - 资金：有没有资金在关注这个股票
    - 成交额/换手率/成交量变化/北向资金流入/主力资金净流入等

## 理论框架

- CAMP 资本资产定价模型
  - 核心想法：一只股票的收益主要来自它承担了市场风险
  - 股票超额收益 = β * 市场超额收益
  - β：股票对大盘的敏感程度
- Fama-French 三因子模型 ：市场因子 市值因子 价值因子
  - 股票超额收益 = 市场因子暴露 + 市值因子暴露 + 价值因子暴露 + 误差
  - 市场因子MKT：市场整体涨跌带来的收益（买股票首先承担整个股市的系统性风险）
  - 市值因子SMB：小市值股票组合收益 - 大市值股票组合收益
  - 接个因子HML：高账面市值比股票组合收益 - 低账面市值比股票组合收益
  - 加入了小盘效应和价值效应
- Fama_French五因子模型（发现三个不够）
  - 盈利能力因子RMW
  - 投资因子CMA
  - 加入公司质量和资本配置能力

## 两个指标

- IC 信息系数 

  - 看因子分数和未来收益有没有相关性

  - 某一天所有股票的因子值 和 这些股票未来一段时间收益率 之间的相关系数

  - | IC     | 含义                           |
    | ------ | ------------------------------ |
    | > 0    | 因子方向大体正确               |
    | = 0    | 因子和未来收益没什么关系       |
    | < 0    | 因子方向可能反了               |
    | > 0.03 | 有一点研究价值                 |
    | > 0.05 | 比较不错                       |
    | > 0.10 | 很强，但现实中要小心是否过拟合 |
    |        |                                |

- 分层回测

  - 如果真按照这个因子选股票，高分组能不能跑赢低分组

    - 选个日期计算所有股票的因子值

    - 按照从高到低排序后分组

    - 观察每组的未来一段时间收益

      单调性越明显因子越好
